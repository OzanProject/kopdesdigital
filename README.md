## ğŸ“¦ Instalasi (Local Development)

1. **Clone Repository**
   ```bash
   git clone https://github.com/OzanProject/kopdesdigital.git
   cd kopdesdigital
   ```

2. **Install Dependencies**
   ```bash
   composer install
   npm install && npm run build
   ```

3. **Environment Setup**
   Copy file `.env.example` ke `.env` dan atur database serta konfigurasi Midtrans Anda.
   ```bash
   cp .env.example .env
   php artisan key:generate
   ```

4. **Database Migration & Seed**
   ```bash
   php artisan migrate --seed
   ```
   *Seeder akan membuat akun Super Admin default dan Paket Langganan dasar.*

5. **Jalankan Validator & Scheduler**
   Untuk fitur Auto-Downgrade, pastikan cron job berjalan:
   ```bash
   php artisan schedule:work
   ```

6. **Serve Application**
   ```bash
   php artisan serve
   ```
