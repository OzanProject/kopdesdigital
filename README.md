## üåü Fitur Unggulan (Core Features)

### 1. Arsitektur Multi-Tenant
- **Subdomain System**: Setiap klien koperasi mendapatkan subdomain unik (contoh: `koperasimaju.kopdes.com`).
- **Data Isolation**: Database dan data antar koperasi terpisah sepenuhnya demi keamanan privasi.
- **Easy Onboarding**: Registrasi koperasi baru hanya butuh waktu kurang dari 1 menit.

### 2. Manajemen Langganan (Subscription) üíé
- **Varian Paket Fleksibel**:
  - Dukungan paket Bulanan (30 hari) & Tahunan (365 hari).
  - **Unlimited Package**: Atur batas member/admin menjadi `0` untuk Tak Terbatas.
  - **Free Tier**: Dukungan paket gratis untuk masa percobaan.
- **Auto-Downgrade System**:
  - Scheduler otomatis mengecek masa aktif setiap hari.
  - Jika langganan habis, akun otomatis turun ke paket Free (tidak langsung diblokir).
- **Payment Gateway**: Terintegrasi dengan **Midtrans** untuk pembayaran otomatis.

### 3. Super Admin Panel (Owner) üëë
Kontrol penuh atas bisnis SaaS Anda:
- **Dashboard Realtime**: Statistik pendapatan, jumlah tenant, dan tiket support.
- **Landing Page CMS**: Edit konten website depan (Hero, Fitur, Harga, FAQ, Testimoni) tanpa coding.
- **Support Ticket System**: Kelola keluhan dan bantuan teknis dari para Tenant.
- **Knowledge Base (Pusat Bantuan)**: Tulis artikel panduan untuk user agar mengurangi beban CS.
- **Backup & Restore**: Fitur backup database otomatis (Local & Cloud) dengan opsi *One-Click Restore*.
- **Invoice Manager**: Pantau seluruh transaksi masuk.

### 4. Tenant Panel (Admin Koperasi) üè¢
Fitur operasional lengkap untuk pengurus koperasi:
- **Manajemen Nasabah**:
  - CRUD Data Anggota dengan foto profil.
  - **Smart Search**: Pencarian cepat berdasarkan Nama/No Anggota.
  - Export/Import Data Excel.
- **Manajemen Pengguna (Petugas)**: Atur hak akses staf koperasi.
- **Laporan & Rekap**: Laporan bulanan otomatis.
- **Upgrade Paket Mandiri**: Tenant bisa upgrade/perpanjang layanan dan memasukkan Kode Promo.

### 5. Keamanan & Teknologi üîí
- **Laravel 12 Framework**: Terdepan dalam performa dan keamanan.
- **Role-Based Access Control (RBAC)**: Pembagian hak akses yang ketat (Super Admin vs Admin Koperasi vs Staff).
- **Secure URL**: Proteksi manipulasi parameter URL pada form registrasi.
- **Auto Validation**: Validasi data input yang berlapis.

## üì¶ Instalasi (Khusus Pemilik Lisensi)

> ‚ö†Ô∏è **PENTING**: Anda memerlukan `LICENSE_KEY` yang valid untuk menjalankan aplikasi ini. Key didapatkan setelah pembayaran.

1. **Clone Repository / Download ZIP**
   (Link Download hanya diberikan kepada pembeli resmi)
   ```bash
   git clone https://github.com/OzanProject/kopdesdigital.git
   cd kopdesdigital
   ```

2. **Setup Dependencies & Environment**
   ```bash
   composer install
   npm install && npm run build
   cp .env.example .env
   # Masukkan Purchase Code Anda di .env
   # PURCHASE_CODE=xxxxxxxx-xxxx-xxxx
   ```

3. **Database Migration**
   ```bash
   php artisan migrate --seed --force
   ```
